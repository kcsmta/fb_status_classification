Bộ phần mềm mã hóa
3 công cụ này sẽ giúp bạn bảo vệ dữ liệu cá nhân lưu ở file và folder trên ổ cứng cũng như đảm bảo những thông tin nhạy cảm mà bạn trao đổi qua e-mail, hay khi duyệt web sẽ không bị kẻ khác xâm nhập.
1. Nếu bạn chưa dùng phiên bản mới nhất của Internet Explorer, công cụ High Encryption Pack sẽ đem lại cho trình duyệt web này tính năng mã hóa 128 bit, mức bảo vệ cao nhất có thể có trên mạng. Đây sẽ là một bức rào phòng vệ hữu ích cho tất cả những giao dịch cá nhân trực tuyến của bạn. Để tải miễn phí High Encryption Pack, hãy bấm vào tên phiên bản Internet Explorer bạn đang dùng trong danh sách dưới đây:
- Internet Explorer 5.01.
- Internet Explorer 5 cho Windows 95, Windows 98 và Windows NT 4.
- Internet Explorer 5 cho Windows 3.1 và Windows NT 3.51.
- Internet Explorer 5 cho Alpha.
- Internet Explorer 4 cho Windows 95, Windows 98 và Windows NT 4.
- Internet Explorer 4 cho Windows 3.1 và Windows NT 3.51.
- Internet Explorer 4 cho Alpha.
2. Chương trình ABI-Coder cho phép bạn mã hóa các file và folder với thuật toán mã hóa Blowfish 448-bit và 3DES 168-bit. Bạn sẽ có thể tạo ra các file tự động giải mã, do đó bạn có thể gửi hoặc truyền các file mã hóa tới những máy tính không cài phần mềm này. Nó thậm chí còn thay đổi cả biểu tượng của những file mã hóa để bạn dễ dàng nhận ra file nào được bảo mật. Để tải miễn phíABI-Coder, bấm vào đây.
3. Nếu bạn cảm thấy lo ngại luôn có một cặp mắt nào đó xem trộm e-mail của mình thì hãy thử phần mềm aCrypt+. Công cụ này cho phép bạn gửi e-mail một cách an toàn với những phần đính kèm được bảo mật cao và được chuyển thành các file thực thi tự giải nén. Người nhận không cần bất cứ phần mềm nào khác để giải mã.
Để lập mã cho e-mail, hãy kéo và đưa file hoặc cấu trúc thư mục đã chọn vào một gói trong aCrypt+, nhập password do bạn tự chọn và rồi bỏ tệp nén tự giải mã này vào trong thư. Còn một số cách dùng aCrypt+ khác như duyệt chọn (browse) các file hoặc folder mà bạn muốn bảo mật hoặc dùng cách tích hợp aCrypt+ vào mục "Send To" trên Desktop của Internet Explorer. Để tải miễn phíaCrypt+, bấm vào đây.
